# 飞牛捷径
此应用可以增强飞牛系统各应用之间的联动和自动化，搭配万能编辑器进行使用
- ⚙️例如在文件管理器空白处右击，即可打开万能编辑器，并且编辑器自动切换到对应目录下
- ✨ 更多联动捷径等你来发现
